## Projekt herunterladen & starten

### Option 1: ZIP Download (kein Git nötig)

1. Klicke oben auf den grünen **Code** Button
2. Wähle **Download ZIP**
3. Entpacke das ZIP auf deinem Rechner
4. Öffne den gewünschten Projektordner (z.B. `Unity`) in der jeweiligen Engine

### Option 2: Mit Git klonen

```bash
git clone https://github.com/ProjectMakersDE/YouTube-Tutorials.git
```

Danach den gewünschten Projektordner in der Engine öffnen.

### Unity Projekt öffnen

1. Öffne den **Unity Hub**
2. Klicke auf **Open** bzw. **Add project from disk**
3. Navigiere zum `Unity` Ordner in diesem Repository
4. Unity lädt das Projekt dann automatisch. Beim ersten Mal kann das etwas dauern
